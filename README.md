# TestingSystem

«TestingSystem» или «Система тестирования студентов» представляет собой дипломный проект по направлению «Программная инженерия 09.03.04» 2021 года.

Для работы проекта необходимо выполнить команды:

- composer install;
- php artisan migrate;
- php artisan db:seed;
- npm install;
- npm run prod.
